# About This Textbook

Welcome to the **AI-Native Robotics Textbook**, a comprehensive learning resource designed to introduce students and enthusiasts to the exciting world of Physical AI and modern robotics.

## What You'll Learn

This textbook covers six essential topics in AI-native robotics:

1. **Introduction to Physical AI** - Understanding embodied intelligence and how robots interact with the physical world
2. **ROS 2 Fundamentals** - Learning the Robot Operating System 2, the nervous system of modern robots
3. **Gazebo & Unity Simulation** - Mastering digital twins and simulation environments for safe robot development
4. **NVIDIA Isaac Platform** - Exploring perception, SLAM, and navigation in advanced robotics platforms
5. **Vision-Language-Action Systems** - Discovering how LLMs enable conversational robotics
6. **Humanoid Capstone Project** - Applying all concepts in a comprehensive simulated humanoid robot project

## About This Project

This textbook was created as part of **Hackathon 2025**, with the goal of making AI-native robotics education accessible to learners worldwide. It combines theoretical concepts with practical insights, using clear explanations and real-world examples.

## How to Use This Book

Start with Lesson 01 and progress sequentially through each module. Each lesson includes:
- Learning objectives
- Core concepts and explanations
- Key terminology
- Summary and takeaways

The capstone project in Lesson 06 brings together everything you've learned into one integrated system.

## Get Started

Ready to begin your journey into AI-native robotics? Head to [Lesson 01](/docs/lesson-01) to get started!
